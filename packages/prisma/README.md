# @stateledger/prisma

Prisma adapter for [stateledger](https://github.com/enowdivine/stateledger).

> Early WIP. Public API not yet stable. Do not use in production.

## Install (once published)

```
pnpm add @stateledger/core @stateledger/prisma
```

`@prisma/client` is a peer dependency — bring your own.
